Transition Design Lab is committed to making its website accessible in accordance with Article 47 of Law No. 2005-102 of February 11, 2005.

This accessibility statement applies to the website transitiondesign.labo.cyu.fr.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Establishment of this Accessibility Statement slug: >- establishment-of-this-accessibility-statement ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

This statement was established on 05/06/2024.

Technologies used for the website's development:

  • HTML5
  • CSS
  • JavaScript
  • Hugo

User agents and assistive technologies used to verify accessibility:

  • NVDA
  • VoiceOver

Accessibility verification was conducted through manual testing, assisted by the following tools:

  • Accessibility Insights for Web
  • ArcToolkit
  • Assistant RGAA
  • Axe DevTool
  • Color Contrast Analyser
  • eAccessibility (PDF accessibility checker)
  • Component Inspector
  • Web Developer Toolbar
  • WCAG Contrast Checker
  • W3C HTML Validator
notes: >- alt: >- credit: >- - kind: block template: title title: >- Testing Environment slug: >- testing-environment ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Content rendering verifications were conducted based on the reference combination provided by the RGAA baseline, with the following versions:

  • On MacOS computer with Google Chrome and VoiceOver
  • On MacOS computer with Safari and VoiceOver
  • On Windows computer with Firefox and NVDA
  • On Android mobile with Google Chrome and TalkBack

If you have reported an accessibility issue to the website manager that prevents you from accessing content or one of the portal's services and have not received a satisfactory response, you have the right to:

  • Send a message to the Defender of Rights - external link (online form);
  • Contact the delegate of the Defender of Rights in your region - external link;
  • Send a postal mail (free, no stamp required) to this address: Défenseur des droits - Libre réponse 71120 - 75342 Paris CEDEX 07.
